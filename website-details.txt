# Personal Developer Portfolio Website PRD

## Project Overview

Build a premium, modern, responsive portfolio website for a Software Engineer and Freelancer.

The website should look like it belongs to a senior developer who works with startups and international clients. It should focus on converting visitors into clients rather than simply showcasing projects.

Design Inspiration:
- Apple
- Vercel
- Stripe
- Linear
- Framer
- Raycast

Style:
- Clean
- Minimal
- Premium
- Fast
- Modern
- Professional

The website should feel elegant with smooth animations, large typography, lots of whitespace, glassmorphism where appropriate, and subtle gradients.
